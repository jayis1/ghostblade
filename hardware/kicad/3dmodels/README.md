# GhostBlade — 3D Model References

These are industry-standard packages — obtain models from manufacturers or
parametric STEP generators.

## Parametric STEP Generation

For packages without manufacturer-provided STEP files, use KiCad's
parametric 3D model generator or FreeCAD with the StepUp plugin:

```bash
# Generate QFN STEP models using KiCad StepUp (FreeCAD macro)
# Install: pip3 install kicad-stepup

# Example: Generate QFN-60-0.4mm_7x7mm for RP2350B
ksu_qfn --pitch 0.4 --pins 60 --body_x 7 --body_y 7 --pad_width 0.2 --pad_length 0.85

# Example: Generate FCBGA-732 for RK3576
ksu_bga --pitch 0.65 --rows 26 --cols 28 --body_x 23 --body_y 23 --ball_dia 0.35
```

## Board Outline STEP

The complete board outline STEP model for mechanical integration:

```bash
# Export board outline from KiCad PCB
kicad-cli pcb export step --output ghostblade-board.step ghostblade.kicad_pcb
```

This produces `ghostblade-board.step` suitable for mechanical CAD integration
(enclosure design, thermal simulation, etc.).

## Enclosure Integration Notes

- Board dimensions: 130mm x 68mm x 1.6mm (PCB only)
- Tallest component: M.2 socket (~4.5mm above PCB)
- RF shielding can height: ~3mm above PCB
- Recommended enclosure internal clearance: 8mm minimum above PCB
- SMA connectors protrude ~12mm from board edge
- Board corner radius: 2mm (for ergonomic handheld enclosure)
